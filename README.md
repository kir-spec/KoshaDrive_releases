## О программе

KoshaDrive: Ваш персональный облачный диск

KoshaDrive помогает использовать ваш аккаунт Telegram как облачное хранилище, предоставляя удобный интерфейс для управления файлами.

Возможности программы:

Свобода действий

Инфраструктура Telegram предоставляет значительные ресурсы для хранения данных, а KoshaDrive делает их управляемыми.

Загружайте видео в высоком разрешении, образы дисков и архивы.

Храните документы, фотографии и резервные копии в одном месте.

Управляйте большими объемами данных без ограничений со стороны программы.

Организация файлов

Больше не нужно искать файлы в ленте сообщений. Виртуальная файловая система предоставляет следующие возможности:

Привычный интерфейс: работайте с файлами как в обычном проводнике Windows.

Древовидная структура: создавайте вложенные папки для удобного порядка.

Управление: добавляйте текстовые комментарии к файлам и выбирайте иконки для папок.

Приватность

Ваши файлы хранятся на серверах Telegram. Приложение спроектировано так, что доступ к структуре данных и файлам осуществляете только вы через свой аккаунт.

Интеллектуальная загрузка

Алгоритмы автоматически управляют фоновой загрузкой. При перетаскивании папки программа распределяет данные и загружает их в облако.

Работа в одном окне

Вы можете отправлять файлы своим контактам напрямую из интерфейса KoshaDrive. Встроенный чат позволяет делиться данными без необходимости открывать основное приложение мессенджера.

KoshaDrive позволяет удобно хранить файлы и организовывать их так, как вам нужно.

Связь с разработчиком - kvf_soft@mail.ru

Первый запуск и вход через Telegram

Шаг 8. Лицензионное соглашение

При первом запуске ознакомьтесь с лицензионным соглашением. Установите флажок «Я прочитал(а) и принимаю условия лицензионного соглашения», после чего станет доступна кнопка «Принять».

Скрин 08

Шаг 9. Окно с рекомендацией использовать свои API Telegram

Программа рекомендует использовать собственные API ID и API Hash Telegram. Для настройки можно нажать «Открыть "Мои API"». Если хотите продолжить сразу, используйте кнопку «Продолжить без гарантий».

Скрин 09

Шаг 10. Авторизация в Telegram

Войти можно по QR-коду или по номеру телефона. При необходимости откройте «Прокси», настройте и нажмите «Далее».

Скрин 10

Шаг 11. Сообщение о коде авторизации

Если Telegram отправил код подтверждения на другое устройство, программа покажет отдельное уведомление. Нажмите «OK» и проверьте код в приложении Telegram на другом устройстве.

Скрин 11

Активация лицензии

Шаг 12. Сообщение об отсутствии активной лицензии

После входа может открыться главное окно с сообщением «Отсутствует активная лицензия». Для продолжения нажмите «Перейти в настройки лицензии».

Скрин 12

Шаг 13. Раздел «Лицензия» в настройках

В настройках откройте раздел «Лицензия». Здесь можно купить лицензию, активировать ключ или включить демо-режим на 14 дней.

Скрин 13

Шаг 14. Ввод лицензионного ключа

Для получения лицензионного ключа свяжитесь с разработчиком через раздел «Справка» → «Написать разработчику» или по почте kvf_soft@mail.ru . Скопируйте ваш HWID в окне активации и отправьте его разработчику. В ответ вы получите лицензионный ключ. В письме укажите дату, время платежа и скрин квитанции об плате лицензии. В комментарии к банковскому переводу обязательно укажите ваш телеграм. Вставьте полученный ключ в поле активации и нажмите «АКТИВИРОВАТЬ ПРОГРАММУ». Если ключа пока нет, вы можете включить демо-режим на 14 дней..

Скрин 14

Шаг 15. Подтверждение успешной активации

После успешной активации появится сообщение о том, что для применения изменений программу нужно перезапустить. Нажмите «OK».

Скрин 15

Шаг 16. Проверка статуса лицензии

После повторного запуска откройте раздел лицензии ещё раз и убедитесь, что отображается статус «ЛИЦЕНЗИЯ АКТИВНА».

Скрин 16

Связь с разработчиком и уведомления

Шаг 17. Раздел «Справка»

В разделе «Справка» можно открыть полное руководство, увидеть адрес поддержки и при необходимости написать разработчику. Для обращения используйте кнопку «Написать разработчику».

Скрин 17

Шаг 18. Уведомление об ответе разработчика

Если разработчик ответит на ваше сообщение, в интерфейсе появится уведомление «Вам ответил разработчик». Нажмите на уведомление, чтобы прочитать ответ.

Скрин 18

Итог: для нормальной работы программы последовательно пройдите установку, вход через Telegram и активацию лицензии. После активации перезапустите KoshaDrive и проверьте, что статус лицензии стал активным.
